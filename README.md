# WorldCupDNA Backend API

Live API: https://worldcupdna-backend.onrender.com

Docs: https://worldcupdna-backend.onrender.com/docs

## Stack
- FastAPI (Python)
- PostgreSQL (Neontech)
- Deployed on Render

## Endpoints
- GET /matches
- GET /venues
- GET /leaderboard
- GET /predictions
- POST /predictions
- GET /profiles/{id}
- POST /profiles
